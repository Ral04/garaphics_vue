Описание

Приложение создает график основываясь на входных данных.

Можно менять не только значения полос графика, но и осевые значения.

Кнопка 'Построить график' на основе данных в текстовом поле.

Кнопка 'Cгенерировать' случайно создает данные и оси.

Инструменты

Для реализации задачи использовались следующие инструменты:

  • Node.js
  • Vue.js CLI
  • Highcharts
  • Sass (scss)
  • Bootstrap
  • Другие...
Задача

Создать web–страницу с текстовым полем ввода и кнопкой “построить график”.

При открытии страницы поле ввода должно содержать редактируемую JSON–строку с данными для построения графика.

При нажатии на кнопку “построить график” на странице должен строиться график стекового распределения данных из текстового поля.
